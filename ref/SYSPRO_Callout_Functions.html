Common callouts    
Show a Task Dialog (TaskDialog, " ", "<Msg><Title></Title><Text></Text></Msg>") Returns the ID of the button clicked followed by the radio button ID and input text, each separated by the TAB character.
Example: 100 101 Hello Worl
Button ID return values: OK=1, Yes=2, No=4, Cancel(Esc)=8, Retry=16, Close=32
Custom button and radio button IDs are numbered from 100 upwards.
Close the application (CloseApp, " ", " ") After the script exits, the application will close.
Get the application title (GetTitle, " ", " ") Returns the application title.
Show a dialog box (ShowDialogBox, "xxxxxxxx", " ") Requires the name of a control to be shown as a dialog box. The option 'Show pane as dialog box' must be checked.
Close a dialog box (CloseDialogBox, "xxxxxxxx", " ") Closes a currently open dialog box.
Show a modal window in SYSPRO Web UI (ShowModalWindow, "modal ID", " ") Requires the ID of the modal window. This callout will only be executed when running in SYSPRO Web UI. See also event 'ModalWindowClose'.
Close a modal window in SYSPRO Web UI (CloseModalWindow, "modal ID", " ") Requires the ID of the modal window. This callout will only be executed when running in SYSPRO Web UI.
Show busy (ShowBusy, " ", " ") Changes the WAIT cursor to indicate the application is busy, or shows a 'busy' marquee in SYSPRO Web UI.
Show ready (ShowReady, " ", " ") Shows the ARROW cursor to indicate the application is ready, or removes the 'busy' marquee in SYSPRO Web UI.
Show the SYSPRO program loading message (ShowSYSPROLoading, " ", " ") Shows the animated SYSPRO gif, typically used to indicate a program is loading. The animation will automatically close after a few seconds.
Show a progress bar (ShowProgressBar, " ", "Title|Message") Shows a progress bar. Typically used for long-running scripts. The progress bar window has a Title and a Message.
Cancel a progress bar (CancelProgressBar, " ", " ") Cancels the progress bar.
Refresh a progress bar (RefreshProgressBar, " ", "new message") Refreshes the progress bar message.
Get company Setup options (GetSetupOptions, " ", "key") Returns company Setup options (by module) as a XML string. The KEY is the module name (such as ARS, APS etc.). You will be shown a list of module names to select from.
Set a variable (SetVariable, "variable name", string) Sets a named variable string. These variables persist for the duration of the run of the application. A variable name cannot exceed 20 characters long. The string may be of any length.
Get a variable (GetVariable, "variable name", " ") Returns a named variable string.
Set a global variable (SetGlobalVariable, "variable name", string) Sets a named global variable string. These variables persist for the duration of SYSPRO. A variable name cannot exceed 20 characters long. The string may be of any length.
It is recommended to prefix your variable name with your application name or part thereof to uniquely identify it.
Get a global variable (GetGlobalVariable, "variable name", " ") Returns a named global variable string.
Get application parameter (GetAppParameter, " ", " ") Returns the value that was passed as a parameter to the application.
Send email (Email, " ", EmailXML) Send an email with optional attachment.
General syntax
<Email><Subject> </Subject><To> </To><Cc/><Message> </Message><Attachment> </Attachment></Email>
Write to Tracer (WriteToTracer, " ", "tracer message") Write a record to the Application Designer Tracer facility.
Refresh a tile in a Web View (RefreshTile, " ", "tile name") Forces a named tile to be refreshed. Leave blank if all tiles are to be refreshed.
Optimize SYSPRO Web UI performance (BeginWebViewUpdate, " ", " ") Prevents any callouts being sent to SYSPRO Web UI until the 'EndWebViewUpdate' callout.
This mechanism can improve the performance of refreshing a Web View
The callout will be ignored if not in SYSPRO Web UI.
Commit SYSPRO Web UI callouts (EndWebViewUpdate, " ", " ") When the EndWebViewUpdate callout is executed any forms, toolbars or charts that were requiring to be refreshed will now be refreshed.
The callout will be ignored if not in SYSPRO Web UI.
Get a datastore variable (GetDatastoreVariable, " ", "key") Returns the value of the KEY from the secure datastore. If the KEY is invalid then the value returned will begin with ERROR:.
Typically this would be used to retrieve credentials.
Raise an event on a toolbar (RaiseToolbarEvent, "toolbarName", "button ID") Execute a specified toolbar and button ID. This may be useful when exiting an application to execute a button on the calling application.
Return a browse value (ReturnBrowseValue, " ", "value") Return a specified value to the calling program's current field. Note that the application will immediately close.
Get application information (GetAppInfo, " ", " ") Returns the details about the running application.
Xml format:
<AppInfo><Author> </Author><ReleaseVersion> </ReleaseVersion><Notes> </Notes><Category> </Category><Apptitle> </Apptitle> </AppInfo>
Set focus to a tab page in SYSPRO WebUI (SetFocusTabPage, " ", "Tab.Caption") Requires the caption of the tab page. This callout will only be executed when running in SYSPRO Web UI.
Data Source Callouts    
Execute a data source (DataSourceExecute, "data source name", "key,key2,key3...") Executes the data source using the Key(s) and returns the data. Up to 9 keys may be entered (separated by a comma) and will replace the placeholders %key%, %key2%, %key3%... .
"0" will be returned if the data source did not execute successfully.
Refresh data from a data source (DataSourceRefresh, "data source name", "key,key2,key3...") Executes the data source using the Key(s) and refreshes the fields associated with the data source. Up to 9 keys may be entered (separated by a comma) and will replace the placeholders %key%, %key2%, %key3%... .
"0" will be returned if the data source did not execute successfully.
Execute a data source with SQL statement (DataSourceExecuteSQL, "data source name", SQLStatement) Executes the custom data source using the supplied SQL statement and returns the data.
Execute a data source with document (DataSourceExecuteXMLDoc, "data source name", xmlDocument) Executes the transactional Business Object data source using the supplied XML document string and returns the data.
"0" will be returned if the data source did not execute successfully.
Get the XML document (DataSourceGetXMLDoc, "data source name", " ") Returns the content of the XML document for a transactional business object. While you can get at this document content in the OnBeforeTransaction event, you may need to get and view the content before executing the data source.
Validate a KEY value (DataSourceValidateKey, "tableName or eventID", KEY + "|xml node|%key% not on file") Returns a status indicating a KEY is valid.
The KEY can be validated against a TableName (such as ArCustomer) or an EventID (such as 38000 indicating the Customer key)
Return values can be DataSourceKeyValid, DataSourceKeyInvalid,"{1}" (invalid table name) or "{2}" (invalid eventID).
Optionally you can supply the XML node to be returned from the TableName, in which case it will be the XML node v
Change the type of transaction (DataSourceSetupType, "data source name", "xxxxxx") Changes the type of transaction for a data source associated with a transaction 'Setup' Business Object. The type (Add, Update or Delete) can be selected from the popup list shown.
Note that this changes the type for the duration of the applicaton.
Set the URL for a Web Service (DataSourceWebSetURL, "data source name", URL) Sets the URL for a web service data source.
Note that this changes the URL for the duration of the applicaton.
Set the body content for a Web Service (DataSourceWebSetBody, "data source name", BodyContent) Sets the body content for a web service data source.
Note that this changes the Body value for the duration of the applicaton.
Set a header for a Web Service (DataSourceWebSetHttpHeader, "data source name", "header key|value") Sets a header key and value for a web service data source. If the header key does not exist it will be added to the list of headers.
Note that this changes the Header for the duration of the applicaton.
Get the properties for a Web Service (DataSourceWebGetProperties, "data source name", " ") Returns the properties of the data source as an XML string.
Get the URL for a Web Service (DataSourceWebGetURL, "data source name", " ") Returns just the URL of the web service.

Get all records in a list view (ListviewGetRecords, "xxxxxxxx", "RowIndex=false") Returns a zero-based 2-dimensional array of rows and columns. The cell values in each row are separated by a TAB delimiter. If the parameter RowIndex is set to true then the first cell value will contain the row index.
Get all 'checked' records in a list view (ListviewGetCheckedRecords, "xxxxxxxx", "nnn|RowIndex=false") Returns a zero-based 2-dimensional array of rows and columns. Returns only records where the check box in the specified column is marked as 'checked'.
The cell values in each row are separated by a TAB delimiter. If the parameter RowIndex is set to true then the first cell value will contain the row index.
Get all 'selected' records in a list view (ListviewGetSelectedRecords, "xxxxxxxx", "RowIndex=false") Returns a zero-based 2-dimensional array of rows and columns. Returns only records that are selected.
The cell values in each row are separated by a TAB delimiter. If the parameter RowIndex is set to true then the first cell value will contain the row index
Note that this callout will not work in SYSPRO Web UI.
Get all 'matched' records in a list view (ListviewGetMatchedRecords, "xxxxxxxx", "textToMatch|nnn|RowIndex=false") Returns a zero-based 2-dimensional array of rows and columns. Returns only records where a cell matches the specified text in a specified column.
The cell values in each row are separated by a TAB delimiter. If the parameter RowIndex is set to true then the first cell value will contain the row index.
Get all 'sorted' records in a list view (ListviewGetSortedRecords, "xxxxxxxx", "RowIndex=false") Returns zero-based a 2-dimensional array of rows and columns. Returns the records in the current sorted order.
The cell values in each row are separated by a TAB delimiter. If the parameter RowIndex is set to true then the first cell value will contain the row index.
Get all 'changed' records in a list view (ListviewGetChangedRecords, "xxxxxxxx", "RowIndex=false") Returns a zero-based 2-dimensional array of rows and columns. Returns only records that have changed.
The cell values in each row are separated by a TAB delimiter. If the parameter RowIndex is set to true then the first cell value will contain the row index.
Get all child records in a list view (ListviewGetChildRecords, "xxxxxxxx", "nnnn") Returns a zero-based 2-dimensional array of rows and columns. Returns the child records of a specified parent row.
Get records as XML (ListviewGetRecordsAsXML, "xxxxxxxx", " ") Returns the records in XML format. The column captions are used as the XML element names.
